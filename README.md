# המשחק המדהים בעולם  🏆

## התקנה

### 1. התקן תלויות
```bash
npm install
```

### 2. הגדר משתני סביבה
העתק את `.env.example` ל-`.env` ומלא את הפרטים:
```
VITE_SUPABASE_URL=https://lprysrpcftkwmmsogwkf.supabase.co
VITE_SUPABASE_ANON_KEY=your_anon_key
```

### 3. הרץ מקומית
```bash
npm run dev
```
